# 🎓 UIU Connect+

A professional social networking platform for the United International University (UIU) community.  
UIU Connect+ provides a secure, feature-rich environment where students, faculty, alumni, and staff can connect, collaborate, and share knowledge.

### Why UIU Connect+?
- **University-Focused:** Tailored for academic communities  
- **Role-Based Access:** Students, Faculty, Alumni, Staff, Club Forums  
- **Content Moderation:** Admin approval system ensures quality content  
- **Professional Design:** Modern, responsive UI with glass-morphism effects  
- **Secure:** Built with security best practices and input validation  

---

### Features

**👥 User Features**
- Secure registration & login with email verification  
- User profiles with bio, skills, contact info, profile & cover photos  
- Role-based registration: Student, Faculty, Alumni, Staff, Club Forum  
- News feed: create posts with text, images, videos  
- Like, comment, share posts  
- Follow system & activity feed  
- Direct messaging with real-time chat  
- Group messaging & file sharing  
- Online status indicators  

**🎓 Academic Features**
- Course groups organized by department and batch  
- Document sharing (PDF, DOC, PPT) with admin approval  
- Academic notices & announcements  
- Event calendar for university events  
- Job board for internships and career opportunities  

**🛍️ Community Features**
- Marketplace for buying/selling items  
- Event management with RSVP  
- Club groups & forums  
- Search functionality for users, posts, groups  
- Notifications for all activities  

**🔧 Admin Features**
- Dashboard & analytics: users, posts, engagement metrics  
- Real-time monitoring & content metrics by role/department/batch  
- Content moderation: user approvals, post moderation, event/job approvals  
- Bulk approval actions  
- User management: view, filter, ban/unban, role & batch management  
- Content management: posts, events, jobs, groups, documents  
- Notices & announcements: create, schedule, target roles, use templates  

---

### Tech Stack
- **Backend:** PHP 8.0+, MySQL 5.7+, PDO, Sessions  
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript, AJAX  
- **Design:** Custom CSS Framework, Responsive Design, CSS Animations, Google Fonts (Poppins)  
- **Development Tools:** XAMPP, Git, VS Code
